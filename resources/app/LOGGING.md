# Система логирования VibeChat

## Описание

В приложение добавлена система логирования, которая записывает все события в текстовый файл. Логи сохраняются автоматически и помогают отслеживать работу приложения и диагностировать проблемы.

## Расположение файлов логов

Логи сохраняются в папке `logs/` в корневой директории проекта:
```
VibeChat/
  logs/
    vibechat-2024-01-15.txt
    vibechat-2024-01-16.txt
    ...
```

Файлы логов создаются с именем формата: `vibechat-YYYY-MM-DD.txt` (один файл на день, формат .txt).
Папка `logs/` создается автоматически при первом запуске приложения.

## Использование

### В коде

```javascript
import { logger } from './modules/logger.js';

// Информационное сообщение
await logger.info('Пользователь вошел', { email: 'user@example.com' });

// Предупреждение
await logger.warn('Неверный пароль', { email: 'user@example.com' });

// Ошибка
await logger.error('Ошибка подключения', { error: error.message });

// Отладка
await logger.debug('Проверка авторизации', { userId: '123' });
```

### Просмотр логов

1. **В консоли браузера**: Все логи также выводятся в консоль браузера (F12)
2. **В файле**: Откройте файл логов в текстовом редакторе

### Формат логов

```
[2024-01-15T10:30:45.123Z] [INFO] Пользователь вошел
{
  "email": "user@example.com"
}
[2024-01-15T10:30:46.456Z] [ERROR] Ошибка подключения
{
  "error": "Connection timeout"
}
```

## Что логируется

Система автоматически логирует:
- ✅ Запуск приложения
- ✅ Инициализация Firebase
- ✅ Авторизация пользователей (вход/выход/регистрация)
- ✅ Ошибки авторизации
- ✅ Показ/скрытие окон (авторизация, основное приложение)
- ✅ Создание файла логов

## Отключение логирования

Если нужно отключить логирование, закомментируйте вызовы `logger` в коде или модифицируйте модуль `logger.js`.

## Очистка старых логов

Файлы логов накапливаются со временем. Вы можете вручную удалить старые файлы из директории `logs/` или настроить автоматическую очистку (не реализовано по умолчанию).
